## 1. What TapeRoll does

TapeRoll keeps a short rolling recording (about the last 2 minutes) of one
configured voice channel so that members can save the last few seconds as a
shareable clip, on demand, using the `/clip` command or the opt-in "clip that"
voice trigger. Clips are posted to a channel the server chooses and can be set to
auto-delete. See the [Privacy Policy](./PRIVACY.md) for exactly what data is
